Airbnb listings in December 2020

--------------------------------------------------------------------------------
Data Dictionary
--------------------------------------------------------------------------------
 
id: Airbnb listing unique id
name: Listing name
description: Full listing description
host_name: Host name
host_about: Host description
latitude: Latitude
longitude: Longitude
property_type: Property type
room_type: Room type
price; Nightly price
amenities: Included amenities
